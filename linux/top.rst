top命令用于显示所有正在运行而且处于活动状态的实时进程，而且会定期更新显示结果。 [1]_

语法
----

``top -hv | -bcHiSs -d delay -n limit -u|U user | -p pid[,pid] -w [cols]``

u 显示特定用户的进程

z 切换彩色/单色

c 显示进程的绝对路径

d 设置刷新时间

k 杀进程

r 设置进程优先级

h 获取帮助

Shift+W 保存自定义配置
